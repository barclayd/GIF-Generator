# Video to GIF Converter

Web app to convert any video to a GIF. Powered by ``ffmpeg`` and WebAssembly

### How to Set Up Project

```shellscript
git clone https://github.com/barclayd/GIF-Generator.git
cd gif-generator
npm i
```

### How to Start Web App

```shell
npm start
```
